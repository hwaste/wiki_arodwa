## ¿Qué es software libre y la diferencia entre copyleft (GPL), permisivas 
### (MIT/BSD/Apache) y propietario. ¿Bajo qué tipo de licencia está Ubuntu?

El software libre es un tipo de software que permite a cualquier persona usarlo, estudiarlo, modificarlo y compartirlo libremente. Esto significa que el código fuente está disponible para que otros puedan revisarlo o mejorarlo. La idea principal del software libre no es que sea gratis, sino que entregue libertad a los usuarios para utilizarlo y adaptarlo a sus necesidades.

Existen distintos tipos de licencias para distribuir software:

Licencias Copyleft (GPL): permiten usar, modificar y distribuir el software, pero obligan a que cualquier versión modificada también se distribuya bajo la misma licencia GPL. De esta forma, las mejoras realizadas por otras personas también permanecen como software libre.
Licencias permisivas (MIT, BSD y Apache): también permiten usar, modificar y distribuir el software, incluso con fines comerciales. La diferencia es que no obligan a publicar las modificaciones bajo la misma licencia, por lo que el código puede incorporarse a proyectos de software propietario. La licencia Apache, además, entrega protección adicional relacionada con patentes.
Software propietario: es aquel cuyo código fuente no está disponible para el público. Los usuarios pueden utilizar el programa según las condiciones establecidas por el fabricante, pero normalmente no pueden modificarlo ni redistribuirlo sin autorización.

En el caso de Ubuntu, es un sistema operativo de software libre. La mayor parte de sus componentes se distribuyen bajo la Licencia Pública General GNU (GPL), aunque también incluye programas que utilizan otras licencias libres, como MIT, BSD y Apache, dependiendo del proyecto. Por ello, Ubuntu se considera una distribución de software libre basada principalmente en licencias copyleft, especialmente la GPL.
